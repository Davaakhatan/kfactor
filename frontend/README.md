# XFactor Frontend - Thin-Slice Prototype

Minimalist, modern UI components for the viral growth system.

## Design Principles

- **Minimalist**: Clean, uncluttered interfaces
- **Modern**: Contemporary design patterns
- **Focused**: Clear CTAs and purpose
- **Accessible**: Proper contrast and readability

## Components

- **PresenceIndicator**: Shows "X peers practicing now"
- **Leaderboard**: Top performers list
- **ShareCard**: Results sharing with CTAs
- **ChallengeInvite**: Invite friends to challenges
- **ActivityFeed**: Recent achievements and activity
- **ResultsShareSurface**: Complete results sharing interface
- **CohortRoom**: Virtual study group rooms

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)

## Design System

- **Colors**: Primary blue, neutral grays
- **Typography**: Inter font family
- **Spacing**: Consistent 4px grid
- **Borders**: Subtle gray borders
- **Shadows**: Minimal, for depth only
