# Privacy Policy Updates
## XFactor Viral Growth System

This document outlines the privacy policy updates required for the XFactor Viral Growth System.

---

## Section 1: Data Collection

### New Data Points

**Viral Growth Features:**
- Invite codes and referral links
- Challenge participation data
- Social interaction data (anonymized)
- Presence indicators (aggregated)

**Session Intelligence:**
- Session transcriptions (with consent)
- Learning summaries (FERPA protected)
- Skill gap analyses (FERPA protected)

### Data Minimization

- Only necessary data collected for viral features
- No third-party data sharing
- Aggregated analytics only
- Fully anonymized for users under 13

---

## Section 2: Data Sharing

### Viral Loops

**What We Share:**
- Anonymous achievements (with consent)
- Aggregated statistics
- Challenge invitations (to parent-approved contacts)

**What We Don't Share:**
- Personal information (email, phone, name)
- Grades or test scores (without explicit consent)
- Educational records (FERPA protected)
- Location data

### Social Features

**What We Share:**
- Aggregated presence counts (e.g., "28 peers practicing now")
- Anonymous leaderboard rankings
- Anonymized activity feeds

**What We Don't Share:**
- Individual student identities
- Personal information
- Educational records

---

## Section 3: COPPA Compliance

### Users Under 13

**Parental Consent Required For:**
- Participating in viral loops
- Sharing results or achievements
- Joining cohort rooms
- Social features participation

**Automatic Protections:**
- All PII automatically redacted
- Anonymous display only
- No third-party sharing
- Privacy-safe defaults

**Parent Rights:**
- Review all data collected
- Request data deletion
- Opt-out of data collection
- Manage privacy settings

---

## Section 4: FERPA Compliance

### Educational Records

**Protected Information:**
- Test scores and grades
- Learning progress
- Diagnostic results
- Session transcripts and summaries

**Sharing Restrictions:**
- No sharing without consent
- Only with parent-approved contacts
- Fully anonymized when shared
- Secure storage and encryption

**Parent/Student Rights:**
- Access to all records
- Request corrections
- Opt-out of directory information
- Request deletion (per retention policy)

---

## Section 5: Consent Management

### Obtaining Consent

**For Users Under 13:**
- Email verification to parent
- Credit card verification ($0 transaction)
- Signed consent form
- Video call verification

**For Users 13+:**
- In-app consent flow
- Clear language and options
- Opt-in for sharing features

### Managing Consent

**Consent Dashboard:**
- View current consents
- Revoke consent at any time
- Update privacy preferences
- Manage sharing settings

---

## Section 6: Data Security

### Technical Safeguards

- **Encryption**: TLS in transit, AES-256 at rest
- **Access Controls**: Role-based, multi-factor authentication
- **Audit Logging**: All access logged and monitored
- **Data Classification**: PII, EDR, Public categories

### Incident Response

- 24-hour breach assessment
- Notification to affected users
- Notification to regulators (if required)
- Remediation plan

---

## Section 7: Data Retention

### Retention Periods

**COPPA (Users Under 13):**
- 3 years after account closure
- Or until user turns 13
- Whichever comes first

**FERPA (Educational Records):**
- Per educational institution requirements
- Typically 5-7 years
- Secure deletion after retention period

**Analytics Data:**
- Aggregated data: Indefinitely (anonymized)
- Individual data: Per retention policy

---

## Section 8: Your Rights

### Access
- Request all data collected about you/your child

- Response within 45 days

### Deletion
- Request deletion at any time
- All data purged within 30 days

### Correction

### Opt-Out
- Opt-out of data collection (while maintaining account)
- Opt-out of directory information
- Opt-out of marketing communications
